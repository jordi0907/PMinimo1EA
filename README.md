# Minimo1 Jordi Rodriguez

# Install Project
1. git clone https://github.com/jordi0907/EA-minimo1
2. Install nodejs, mongodb 

# Execute Backend
1. cd backend
2. Execute "npm install". 
3. Start mongod.
4. Execute "npm run dev"

# Execute Frontend

1. cd frontend 
2. Execute "npm install". 
3. Execute "ng serve"
